# Release 2026-02

### ✨ Nuevas Funcionalidades
![Static Badge](https://img.shields.io/badge/Module-Inventarios-green)

**Copiar Referencias:**

Ahora puedes duplicar referencias existentes de forma rápida y sencilla:

* **Botón de Copia:** En el listado de referencias aparece un botón azul con ícono de copiar (solo para referencias tipo 'R').
* **Proceso Simplificado:** Al hacer clic, se abre una ventana donde solo debes ingresar:
  - El nuevo código para la referencia
  - El nuevo nombre/descripción
* **Copia Completa:** El sistema duplica automáticamente:
  - Todos los atributos configurados
  - Materiales asociados
  - Proveedores
  - Consumos
  - Operaciones
  - Consumos por atributos
* **Acceso Directo:** Al finalizar la copia, el sistema te lleva directamente a editar la nueva referencia creada.
* **Validación en Tiempo Real:** El sistema valida automáticamente que el código ingresado no esté duplicado mientras escribes.

Esta funcionalidad es ideal cuando necesitas crear referencias similares, ahorrando tiempo al evitar configurar todo desde cero.

📖 **[Ver documentación completa sobre Copiar Referencias](../inventarios/maestros/referencias.md#copiar-referencias)**

### 🐞 Bugs Corregidos

**Corrección de Modal de Referencias (Atributos):**

* **Error:** Cuando se cerraba la ventana modal de Referencias, el indicador de carga (rueda giratoria) se quedaba visible. Además, si se volvía a abrir la ventana, el indicador seguía pegado y la ventana no respondía correctamente.
* **Afecta a:** Pedidos, Traslados, Entradas-Salidas, Ventas
* **Solución:** Se mejoró el funcionamiento de la ventana modal para que el indicador de carga desaparezca correctamente, y la ventana funcione sin problemas cada vez que se abre.
