# Inventarios Físicos

La opción **Inventarios Físicos** permite consultar los conteos físicos registrados y revisar el detalle por cada inventario.

## Ruta

**Inventarios → Consultas/Reportes → Inventarios Físicos**

## ¿Qué se puede hacer?

- Consultar el listado de inventarios físicos registrados.
- Validar fecha, bodega, estado y usuario de creación.
- Abrir el detalle de un inventario específico para revisar referencias, atributos y cantidades.

## Pasos de uso

1. Ingresar a la opción **Inventarios Físicos**.
2. Ubicar el registro deseado en la tabla principal.
3. Hacer clic en el botón **Ver Detalle** (ícono de lista).
4. Revisar en la sección inferior el detalle del inventario seleccionado.

## Información mostrada

### Encabezado
- Fecha
- Bodega
- Activo
- Usuario Creación
- Fecha/Hora Creación

### Detalle
- Código Referencia
- Nombre Referencia
- Atributo Principal
- Atributo Secundario
- Cantidad
- Usuario Creación

## Nota

Actualmente la opción está habilitada en modo **solo consulta**.
