# CONFIGURACIÓN

## Compañías - Parámetros de Automatización

Para habilitar integración con base de datos externa (SwiftBots/WhatsApp), en Configuración → Compañías se deben configurar los campos:

- Url Base de Datos Automatización
- ApiKey Base de Datos Automatización

Estos parámetros son usados por Movimientos de Producción para sincronizar automáticamente la información del movimiento en ENTREGA.
