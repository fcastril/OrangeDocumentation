## 📸 Imágenes (img/)

Aquí se guardan todas las capturas de pantalla y diagramas utilizados en la documentación.

### Convenciones de Nombrado

- Usar nombres en **minúsculas con guiones** (kebab-case)
- Ejemplo: ✅ `reporte-general.png` (no ❌ `ReporteGeneral.png`)
- Formato recomendado: `.png` (sin pérdida de calidad)

### Tamaño Recomendado

- **Ancho máximo:** 800px
- **Peso máximo:** 500KB por imagen
- **Resolución:** 96 DPI
