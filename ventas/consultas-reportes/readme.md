### 📋 Reporte de Pedidos

Herramienta para visualizar y analizar el estado de los pedidos con múltiples opciones de agrupación:
- **General:** Pedidos con desglose de líneas
- **Por Cliente:** Pedidos resumidos por cliente
- **Por Referencia:** Pedidos agrupados por producto
